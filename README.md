# 🎮 打砖块游戏 (Breakout Game)

基于 Matter.js 物理引擎的经典打砖块游戏，完美适配手机端触屏操作。

## 🎯 游戏特点

- ✨ 使用 Matter.js 2D 物理引擎，真实物理效果
- 📱 完美适配手机触屏，支持滑动控制挡板
- 🎨 现代化 UI 设计，渐变色彩
- 💫 流畅的 60FPS 动画
- 🏆 计分系统和生命系统

## 🕹️ 游戏操作

### 手机端
- 手指在屏幕上左右滑动控制挡板移动

### 电脑端
- 鼠标左右移动控制挡板

## 🚀 在线游玩

访问 [Vercel 部署链接](https://breakout-game.vercel.app) 即可开始游戏！

## 🛠️ 本地运行

```bash
# 安装依赖
npm install

# 启动本地服务器
npm start
```

然后在浏览器中打开 `http://localhost:3000`

## 🎮 游戏规则

1. 控制挡板反弹小球
2. 击碎所有砖块即可通关
3. 小球掉落会失去一条生命
4. 共有 3 条生命
5. 不同颜色的砖块分数不同（越高分数越高）
